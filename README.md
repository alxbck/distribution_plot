# distribution_plot

### How to launch

Create a virtual environment:
`python -m venv .venv`

Activate it:
`.venv\Scripts\activate`

Install python dependencies:
`pip install -r requirements.txt`

Launch it:
`voila distribution_plot.ipynb`
